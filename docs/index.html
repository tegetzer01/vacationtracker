About

VacationTracker is an Android application that helps users plan vacations and associated excursions. It supports secure login, creation and editing of vacations with validation, optional reminders via system notifications, excursion management, reactive search, and a reporting screen that can export a CSV for sharing.

  • Plan vacations with title, hotel, start/end dates, and reminders
  • Manage excursions tied to a vacation, constrained to its date range
  • Search by title/hotel and generate a CSV itinerary report
