About

VacationTracker is a lightweight Android application that helps users organize and track their travel plans with ease. Designed for devices running Android 8.0 and above, it allows users to log vacations and excursions, set reminders, search itineraries, and export trip data in CSV format. Distributed as an APK, VacationTracker combines intuitive design with secure data handling to support seamless trip planning and reflection.
